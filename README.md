# Лабораторна робота №7
# ТЕМА: Методи кластеризації 
# ХІД РОБОТИ
У ході лабораторної роботи було виконано кластеризацію набору даних Wine методом k-means. Спочатку було імпортовано необхідні бібліотеки та завантажено набір даних. Далі виконано масштабування ознак за допомогою StandardScaler для коректної роботи алгоритму. За допомогою методу ліктя визначено оптимальну кількість кластерів, яка дорівнює 3. Після цього проведено кластеризацію даних методом k-means та отримано мітки кластерів для кожного об’єкта. Якість кластеризації оцінено за допомогою метрик Adjusted Rand Index, Silhouette Score та матриці відповідностей. Отримані результати показали, що алгоритм успішно розділив дані на кластери, які добре відповідають реальним класам набору даних Wine.
